# Changelog

## [1.4.4]
### Added
- Added new hash matching patterns:
  - Support for SHA-224, SHA-3 (224, 256, 384, 512), and MD6 (256/512).
  - Extended matching for CRC32, Adler-32, and GOST hashes.
  - Improved compatibility with non-standard hash lengths.
